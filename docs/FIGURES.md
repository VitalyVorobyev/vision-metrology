# Figures

Deterministic gallery figures are generated by `tools/figures/generate_all.py`.
These scripts are documentation tooling only and are not Rust crate dependencies.

## Regenerate all

```bash
python tools/figures/generate_all.py
```

If needed, set up a local Python environment first:

```bash
cd tools/figures
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python generate_all.py
```

## Figure index

- `docs/fig/morphology_open_close.png`
  - Demonstrates binary morphology (input with noise/holes, open, close).
  - Generator: `figgen.morphology_fig.make_figure()`.

- `docs/fig/pyramid_levels.png`
  - Demonstrates 2x2 mean pyramid levels (`L0..L4`) with drop-odd behavior.
  - Generator: `figgen.pyramid_fig.make_figure()`.

- `docs/fig/edge1d_dog_stripe.png`
  - Demonstrates 1D stripe intensity, DoG response, edge extrema, and center.
  - Generator: `figgen.edge1d_fig.make_figure()`.

- `docs/fig/laser_rows.png`
  - Demonstrates row-wise laser stripe tracking (coarse + ROI + edge-pair idea).
  - Generator: `figgen.laser_fig.make_rows_figure()`.

- `docs/fig/laser_cols.png`
  - Demonstrates column-wise laser stripe tracking on a horizontal stripe scene.
  - Generator: `figgen.laser_fig.make_cols_figure()`.

- `docs/fig/edgels_overlay.png`
  - Demonstrates 2D edgel points and gradient normals over a synthetic scene.
  - Generator: `figgen.edge2d_fig.make_figure()`.

- `docs/fig/contour_graph_ty.png`

  - Generator: `figgen.contour_fig.make_ty_figure()`.

- `docs/fig/contour_graph_loop.png`
  - Demonstrates loop contour component with `LoopAnchor` concept.
  - Generator: `figgen.contour_fig.make_loop_figure()`.
